errorFlash
directive in module App

Description

Muestra en pantalla los mensajes que devuelve la api.

Usage

as attribute
<ANY error-flash
     tipo="{string}"
     mensaje="{string}"
     object="{object}">
   ...
</ANY>

Parameters

ParamTypeDetails
tipostring

tipo(success,danger,warning,info)

mensajestring

mensaje

objectobject

en caso de ser objeto errorFlash.error(data)

Example

flash('success', 'texto del mensaje'); o errorFlash.error(array('success','texto del mensaje'));
